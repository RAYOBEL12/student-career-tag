# 📘 Day 2 – MVP and Functional Modules

🗓️ **Date:** June 19, 2025  
🔨 **Phase:** Planning, Design & Setup  
📁 **Project:** Student Career TAG PWA (Django + React)

---

## 🎯 MVP Goal

The goal of this MVP (Minimum Viable Product) is to enable university students to:

- Register and access the platform
- Input academic data (coursework, exams, GPA)
- Analyze their strengths and weaknesses
- Get career guidance based on academic performance
- Track their goals and future progress

---

## ✅ MVP Functional Modules

### 1. 🔐 Auth Module
- Student registration
- Login/logout system
- (Future) Password recovery and reset

---

### 2. 📥 Academic Data Input Module
- Form for manual entry of:
  - Programme of study
  - Course units
  - Coursework marks
  - Exam scores
- Option to upload CSV or JSON file
- Input validation for GPA ranges

---

### 3. 📊 Performance Analysis Engine
- GPA calculator (course-based and cumulative)
- Identification of strong vs. weak areas
- Display insights (simple charts)

---

### 4. 🤖 Career Recommendation Engine
- Rule-based matching:
  - High GPA in Programming → Software Developer
  - High GPA in Networking → Network Admin
  - Low GPA → Recommend improvement areas
- (Future) AI/ML model to enhance suggestions

---

### 5. 🧭 Career Tracker Dashboard
- Visual feedback (bar charts, timelines)
- Track career interest vs academic trend
- Save goals and compare with progress

---

## 🔭 Future Features (Post-MVP)
- Admin dashboard for managing users
- Lecturer/Advisor module to give feedback
- Notifications and alerts
- Peer comparison (student ranking or cohort trends)
